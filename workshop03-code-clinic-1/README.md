# Workshop 3: Code Clinic 1 - Linux & HPC intro

Some useful resources - Here's a curated guide to the best resources for each topic, tailored for experimental biologists:

---

## Part 1: Command Line Tutorials for Biologists

**1. Software Carpentry** 

🔗 https://swcarpentry.github.io/shell-novice/

Some basic tutorials on file systems, navigation, shell scripts, and loops. 

Also check out this one more tailored to genome bioinformatics (real biological data formats (FASTQ, genome files, etc.)

🔗 https://datacarpentry.org/shell-genomics/

N.B. Software Carpentry is a volunteer project dedicated to teaching basic computing skills to researchers - might be a good starting point to learn some coding!

**2. Introduction to Linux for Bioinformatics — FutureLearn/Wellcome Genome Campus** *(Free to audit)*

🔗 https://www.futurelearn.com/courses/linux-for-bioinformatics

An online course covers Linux basics, Bash scripting, and R — all using biological data case studies.

**3. Bioinformatics 101 by Hadrien Gourle** *(Free)*

🔗 https://www.hadriengourle.com/tutorials/command_line/

A great place to learn the command line (alongside file formats and standard programs etc.) but more in the context of next-generation sequencing data analysis — very biology-specific examples.

**4. "Ten Simple Rules for Getting Started with Command-Line Bioinformatics" (PLoS Comput Biol)**

🔗 https://pmc.ncbi.nlm.nih.gov/articles/PMC7891784/

A great paper - not a tutorial per se, but written specifically for researchers with a biology/genetics background stepping into bioinformatics.

## Part 2: Coursera/edX Courses for ML/AI, Python and Bioinformatics

These are ordered from most accessible (biology-first) to more technical:

**1. Bioinformatics Specialization — UC San Diego (Coursera)** 

🔗 https://www.coursera.org/specializations/bioinformatics

**2. Genomic Data Science Specialization — Johns Hopkins (Coursera)** 

🔗 https://www.coursera.org/specializations/genomic-data-science

**3. HarvardX Data Analysis for Life Sciences — edX** *(Free to audit)*

🔗 https://www.edx.org/professional-certificate/harvardx-data-analysis-for-life-sciences

**4. Machine Learning Specialization — Stanford/DeepLearning.AI (Coursera)**

🔗 https://www.coursera.org/specializations/machine-learning-introduction

The gold-standard intro to ML by Andrew Ng. Not biology-specific, but gives a rigorous foundation before tackling bioinformatics-specific applications. 

**5. AI for Medicine Specialization — DeepLearning.AI (Coursera)**

🔗 https://www.coursera.org/specializations/ai-for-medicine

A three-course specialization offering practical experience in applying machine learning to concrete medical problems. Requires some Python familiarity.

**6. Python for Everybody Specialization — University of Michigan (Coursera)**

🔗 https://www.coursera.org/specializations/python

An excellent beginner Python course!

**7.  Biology Meets Programming: Bioinformatics for Beginners — UC San Diego (Coursera)**

🔗 https://www.coursera.org/learn/bioinformatics

On algorithms for solving biological problems, with programming challenges to implement those algorithms in Python.
